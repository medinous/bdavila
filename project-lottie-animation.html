Lottie animation

This project was created for Lendstar, a social finance app for making online payments among friends where you can collect money, divide bills and transfer funds directly from your bank account. To improve the user experience during the payment process, I created an icon animation with Lottie library for when the user successfully completes a money transaction.

01/ Process

I started the animation drawing the shapes in Illustrator, then exported the vector layers into After Effects to build the animation. Using the bodymovin plugin, I exported the full animation into a .json file that works for both iOS and Android. This is the actual standard process to work with Lottie files and you can take a look on how this step-by-step works by watching their own beginner's guide video.

02/ Outcome

The final result was an extremely light animation ready to be implemented on both iOS and Android operational systems, which creates consistency and does not compromise performance over quality. The benefit of this approach is that you can create very interesting and complex animations using a single file translated into code ready to be handled by developers, and the exact same file can be implemented for both iOS and Android with no duplicates needed.
